28. Create trigger to insert data in emp_log table whenever any update of sal in emp table. You can set action as ‘New Salary’. 

DELIMITER
$$

CREATE TRIGGER before_salary_update BEFORE UPDATE ON
emp FOR EACH ROW
BEGIN
INSERT INTO emp_log
SET Action 	= 'New Salary',
New_Salary	= new.Sal,
Emp_id		= old.Empno;

END
$$
